### Week 9 Objectives:
* Apply secure access limits and mitigate risks using IAM Permission Boundary.
* Centrally manage and automate patching and command execution with AWS Systems Manager.
* Collect performance metrics and optimize server configurations (EC2 Right-sizing) with CloudWatch Agent and Compute Optimizer.
* Protect stored data (Encrypt at rest) with AWS KMS, combined with log tracking via CloudTrail and Athena.

### Tasks to be carried out this week:
| Day | Task | Start Date | Completion Date | Reference Material |
| :--- | :--- | :--- | :--- | :--- |
| 2 | **IAM Permission Boundary:** Learn how to prevent privilege escalation. Practice creating restriction policies, granting permissions, and verifying permission limits for an IAM User. | 15/06/2026 | 15/06/2026 | https://000030.awsstudygroup.com/1-introduction/ |
| 3 | **AWS Systems Manager:** Centrally manage resources. Configure Patch Manager and use Run Command to automate software patching and command execution across multiple servers simultaneously. | 16/06/2026 | 16/06/2026 | https://000031.awsstudygroup.com/1-introduction/ |
| 4 | **EC2 Right-sizing:** Install CloudWatch Agent to collect memory metrics. Utilize AWS Resource Optimization and AWS Compute Optimizer to analyze and select the appropriate EC2 configurations. | 17/06/2026 | 17/06/2026 | https://000032.awsstudygroup.com/1-introduction/ |
| 6 | **Encryption with AWS KMS:** Create a Key Management Service (KMS) key and set up encryption to protect data on Amazon S3. Combine with CloudTrail to log API activity and use Athena to query the logs. | 19/06/2026 | 19/06/2026 | https://000033.awsstudygroup.com/1-introduction/ |


### Week 9 Achievements:
* **Access Control Security (IAM):** Successfully understood and applied IAM Permission Boundary, effectively preventing privilege escalation risks in systems with complex user roles.
* **Operational Automation (Systems Manager):** Simplified administration by successfully automating large-scale operational tasks (such as applying patches and running commands), significantly minimizing manual effort.
* **Cost & Performance Optimization (EC2 Right-sizing):** Successfully collected in-depth metrics via CloudWatch Agent and analyzed them using Compute Optimizer to make optimal server configuration decisions, balancing cost and performance.
* **Data Security (KMS & Data logs):** Successfully encrypted data at rest, while establishing a monitoring and tracking system for API calls accessing data, ensuring strict adherence to security compliance standards.
