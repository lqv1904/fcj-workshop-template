### Overall Evaluation

**1. Program Content**  
The program is built on a logical roadmap, starting with fundamental AWS knowledge before moving to practical deployment content. The combination of theory and practice provides learners with the opportunity to immediately apply the knowledge they have acquired. Thanks to this, when entering the group Workshop, the deployment of the assigned infrastructure went more smoothly.

**2. Documentation Quality**  
The study materials and Workshop guides are presented in great detail, with clear illustrations and step-by-step instructions. Dividing the content among members helps accurately define the scope of work and minimizes overlap during the deployment process. Additionally, the official AWS documentation serves as a useful reference for gaining a deeper understanding of each service.

**3. Skills Acquired**  
After 12 weeks of studying and practicing, knowledge regarding identity management with IAM, and network infrastructure design using Amazon VPC, Public/Private Subnets, NAT Gateways, and Security Groups has been significantly strengthened. At the same time, skills in planning, team coordination, and infrastructure handover among members have also improved through the Workshop execution process.

**4. Program Support**  
The AWS Credits provided in the program create favorable conditions for practicing in a real AWS environment without worrying too much about costs. Furthermore, the Workshop sessions along with the instructional documentation system provided effective support during the learning and project deployment process. We also received a lot of support from all the program admins and others.
